Superpowers

  • ✓ Drag-and-drop WYSIWYG page builder — non-technical stakeholders compose pages at runtime without touching code

  • ✓ Benefit Realization Management — track planned vs. actual business outcomes for every initiative

Intelligence (IQ Suite)

  • ✓ Natural Language NLP Module — users query any entity in plain language; translates intent to SQL and renders results as tables or charts
  • ✓ Safe AI Agent Engine — deterministic, observable, reversible automation; AI benefits without hallucination risk
  • ✓ Goal-Seeking Scenarios — propose, evaluate, and select strategies; AI forecasts outcomes and generates action plans
  • ✓ Configurable LLM backends — OpenAI, Watsonx.ai, Ollama, Mistral

Enterprise Governance

  • ✓ Advanced ReBAC — graph-based access control for complex orgs ("user may access a record only if they are a member of the project that owns it")

  • ✓ Compliance Audit Logs — automated Who/What/When tracking for every write; SOC 2 and ISO 27001 ready
